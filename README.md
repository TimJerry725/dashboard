# Zwiz Dashboard

A React + TypeScript application built using the shared Frontend UI System.

## Project Structure

- `src/styles/theme.ts`: Design tokens (colors, typography, spacing).
- `src/components/`: Reusable UI components (Sidebar, Header, PageHeader).
- `src/pages/`: Application pages (Dashboard).
- `src/App.tsx`: Main layout and routing.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
